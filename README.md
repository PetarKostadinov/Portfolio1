# Petar Kostadinov — Portfolio

A responsive personal portfolio presenting my professional experience, technical skills, education, certifications, and selected software projects.

## Live Portfolio

[View the deployed portfolio](https://portfolio1-aya.pages.dev/)

## Highlights

- Responsive layouts for desktop, tablet, mobile, and landscape screens
- Accessible section navigation with active-state tracking
- Professional experience and education timeline
- Technical skills and professional strengths overview
- Selected project links and GitHub references
- Contact form powered by Formspree
- Lightweight implementation with no build step or framework dependency

## Technologies

- HTML5
- CSS3
- JavaScript
- [Boxicons](https://boxicons.com/) for interface icons
- [Formspree](https://formspree.io/) for contact-form delivery

## Run Locally

No installation or build step is required. Clone the repository and serve its root directory with any static web server.

```bash
git clone https://github.com/PetarKostadinov/Portfolio1.git
cd Portfolio1
```

For example, use the Live Server extension in Visual Studio Code, or open the project through another local static server.
